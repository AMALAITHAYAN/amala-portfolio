# Amala Ithayan Portfolio

A premium animated personal portfolio built with Next.js, Tailwind CSS, and Motion.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Highlights

- Animated hero section with floating badges and terminal-style profile card
- Scroll progress indicator
- Tech marquee
- About, services, projects, experience, skills, achievements, and contact sections
- Downloadable resume in `public/amala-ithayan-resume.pdf`
- Responsive dark visual system
